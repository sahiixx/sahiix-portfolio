# Investor Outreach — templates + readiness

## A) Funding readiness checklist (have these before first calls)

**Narrative**
- [ ] One-liner memorized (see FUNDING-NARRATIVE.md §2).
- [ ] 3-tier story crisp: Kernel → OS → Proof.
- [ ] Honest traction line: code shipped + live deployments, not "customers".

**Artifacts (open in the call)**
- [ ] 10–12 slide deck (Problem / Solution / Architecture / Live Systems / Market / Founder / Ask).
- [ ] Portfolio matches the story (it does — Systems Panel added).
- [ ] GitHub `sahiixx/sahiixx` cleaned (READMEs, descriptions, no broken builds).
- [ ] **Systems Panel page live** — the credibility device ("real systems, not slides").

**Live proof (differentiator vs slide-only startups)**
- [ ] SAHIIX OS screens reachable + demoable.
- [ ] NEXUS lead→CRM walkable end-to-end (WhatsApp intake → ranked queue).
- [ ] sahiixx-agency status view showing live vs in-dev.
- [ ] One voice demo (Jarvis) if leading with the edge-agent angle.

**Outreach**
- [ ] Target list of 8–12: UAE/GCC pre-seed already in AI proptech + AI-agent/security.
- [ ] 3–5 warm intros prioritized over cold.
- [ ] Email + LinkedIn DM templates below, personalized per thesis.
